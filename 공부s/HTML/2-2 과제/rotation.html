오늘의 목표는 원의 방정식에서 원환면 유도하기이다.


먼저, 삼각함수의 성질 a²cos²θ + a²sin²θ = a² 관계식을 이용해 매개화된 방정식을 세울 수 있다. 이 때 원의 반지름은 a이다.


x = 1 * cosθ, y = 1 * sinθ 인 매개화를 진행한다고 해보자


이 때, cos²θ + sin²θ = 1 관계식의 cosθ, sinθ를 각각 x, y로 바꾸면 x² + y² = 1 이라는 원의 방정식이 만들어진다.

이 방정식은 원점을 중심으로하고 반지름이 1인 원의 방정식을 표현하는 방법이다.


이러한 원을 x축 +방향으로 3만큼 움직여보자.


현재 원의 중심이 (0, 0)에 있으므로 x축 방향으로 3만큼 움직이면 원의 중심이 (3, 0)이 된다.


즉, y값에는 변화를 주지 않고 x = 3일 때 현재 그래프의 x = 0의 역할을 하도록 해야한다.


x가 3일 때 x에 대한 식은 0이 나와야 하므로 x에 x - 3을 대입한다.


따라서 평행이동한 원의 방정식은 (x-3)² + y² = 1이고 이를 매개화하면 x = cos(t) + 3, y = sin(t)이다.


이제 3차원으로 넘어가 이 원을 y축 방향으로 회전하는 일만 남았다.


원의 방정식의 중심 좌표가 3이므로 a² = 3

z축을 추가한 후, 원점을 중심으로 하고 반지름을 √3으로 하는 원의 방정식을 x-z평면에 그릴 수 있고, 이러한 원의 방정식은 원환면의 중심을 지나게 된다.


x-y평면의 원의 방정식에서 각 점의 x좌표를 반지름으로 하는 원을 그릴 것이므로 x = xcosφ, z = xsinφ, y = sinθ가 된다.


따라서 x = (cosθ + 3)cosφ, z = (cosθ + 3)sinφ, y = sinθ이다. 이를 그래프로 그려보자.

해당 그림에선 y축과 z축이 바뀌어있다.
